# Backlog

## Planned Features

### Timing Modes
Allow the game clock to operate in different modes to support different game formats.
- Examples: count-up vs count-down, period-based timing, overtime modes
- Configurable per game or saved as a preset

### Challonge Live Scoring Integration
Push live score updates back to Challonge as the game progresses, rather than only reading match data from the bracket.
- Automatically report final scores to Challonge when the game ends
- Advance the bracket without needing to manually update Challonge
